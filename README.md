## 📌 Sobre o Projeto

Site profissional desenvolvido para o **Dr. Erigutemberg Meneses**, reunindo sua trajetória acadêmica, atuação profissional e obra literária completa (livros e artigos). O projeto apresenta uma interface moderna, responsiva e de fácil navegação, destacando a tripla expertise do autor.

### 🎯 Objetivos
- Apresentar o perfil profissional de forma elegante e acessível.
- Catalogar toda a produção intelectual (livros e artigos acadêmicos).
- Permitir filtragem dinâmica das obras por categoria.
- Oferecer canais de contato direto para palestras, consultorias e aquisição de obras.

| Tecnologia | Descrição |
